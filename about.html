More about HelloDoc

HelloDoc is an intelligent model built using Machine Learning. You can upload an image of the infected area and HelloDoc classifies your dermatological problem. Currently we have trained this model to classify five diseases

1. Basel cell Cancer

2. Melanoma

3. Squamous cell carcinoma

4. Lentigo

5. Seborrhoeic keratosis

The model will be scaled further to classify more skin related diseases.
